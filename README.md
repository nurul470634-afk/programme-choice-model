Programme Choice Model (MNL)

This project implements a Multinomial Logit (MNL) model using Octave to simulate student programme selection.

#Features
GUI-based input system
Adjustable weights and ratings
Probability calculation
Bar chart visualisation
#How to Run
Open Octave
Run:
programme_choice
Input values and click Calculate
#Output
Displays most preferred programme
Shows probability distribution using bar chart
